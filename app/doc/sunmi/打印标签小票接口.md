
## 一、功能介绍

---

```java
public CanvasApi canvasApi()

```

标签小票绘制整张标签后打印的接口集合，通过接口绘制完整的打印内容后打印，但对绘制内容设计要求更高，其更容易实现图文混排的打印效果，同时打印阶段会自动定位标签位置，所以多用于打印标签、黑标的场景

void initCanvas(BaseStyle style)void renderText(String text, TextStyle style)void renderBarCode(String code, BarcodeStyle style)void renderQrCode(String text, QrStyle style)void renderBitmap(Bitmap bitmap, BitmapStyle style)void renderArea( AreaStyle format)void printCanvas(int count, PrintResult listener)

## 二、使用限制

此接口需要使用者将商米内置打印机类型切换为标签或黑标模式才可以正常使用，对于开发者可使用如下代码跳转到类型设置界面：

```java
Intent intent = new Intent();
intent.setAction("com.sunmi.printer.settings");
intent.setPackage("woyou.aidlservice.jiuiv5");
Bundle bundle = new Bundle();
bundle.putInt("item", 1);
intent.putExtras(bundle);
if(intent.resolveActivity(getPackageManager()) != null) {
   startActivity(intent);
}

```

跳转设置能力需要打印服务6.6.32以上版本；商米内置打印机也可以在热敏票据模式下使用此接口实现热敏票据打印图文混排内容；

## 三、接口说明

### 1. 画布设置

```java
void initCanvas(BaseStyle style)

```

指定打印标签的画布大小、基础格式

使用画布渲染打印内容必须首先初始化要绘制的画布大小，才能确保最终可打印;

**BaseStyle 说明**

| 可用方法 | 方法说明 | 参数说明 | 默认值 |
| --- | --- | --- | --- |
| setWidth | 初始化预渲染的画布宽度 | 可设置大小不超过打印纸宽度 | 默认0像素 |
| setHeight | 初始化预渲染的画布高度 | 可设置大小不超过打印纸高度 | 默认0像素 |
| setPosX | **东纳打印机**绘制区域的相对位置 | 单位像素 | 默认0像素 |
| setPosY | **东纳打印机**绘制区域的相对位置 | 单位像素 | 默认0像素 |
| setRenderColor | **东纳打印机**支持打印内容颜色设置 | 支持黑色和红色墨盒 | 黑色墨盒 |
| setAlign | 不可用 | 不可用 | 无 |


setWidth和setHeight必须指定大小才能进行内容绘制；

**LabelStyle 说明**

| 可用方法 | 方法说明 | 参数说明 | 默认值 |
| --- | --- | --- | --- |
| enableReverse | 标签打印方向设置 | 是否反向 | 打印机默认设置 |
| enableMirror | 标签打印镜像设置 | 是否镜像 | 打印机默认设置 |
| enableBack | 当打印机支持切刀时每次切刀结束时根据此标志决定下一张标签是否回退当打印机不支持切刀时打印结束后若开启了撕纸标志将决定下一次打印是否回退 | 是否开启 | 打印机默认设置 |
| enableTear | 当打印机不支持切刀时决定是否在打印结束后将标签纸进纸到撕纸位置 | 是否开启 | 打印机默认设置 |

| setPosY | 设置文本内容在画布起始纵坐标位置 | 单位像素 | 0 |

| setTextSize | 字体为矢量时指定文本字符大小 字体为非矢量字体文本字符大小固定 | 有效范围6~96像素 | 24 |
| setTextWidthRatio | 指定倍宽大小 | 有效范围0-9 | 0 |
| setTextHeightRatio | 指定倍高大小 | 有效范围0-9 | 0 |
| setWidth | 设置文本限制宽度 | 若设置宽度限制，超过宽度将自动换行 | 不限制 |
| setHeight | 设置文本限制高度 | 若设置高度限制，超过部分将不显示 | 不限制 |
| setAlign | 设置文本内容相对起始坐标的位置 | [见Align](#enum1) | Align.DEFAUL |
| setRotate | 设置文本内容的方向 | [见Rotate](#enum2) | 水平方向 |
| setTextSpace | **内置打印机**设置文本字间距 | 0~100像素 | 0 |
| enableBold | **内置打印机**设置文本加粗 | 开启文本加粗功能 | 不开启 |
| enableUnderline | **内置打印机**设置文本下划线 | 开启文本下划线功能 | 不开启 |
| enableStrikethrough | **内置打印机**设置文本删除线 | 开启文本删除线功能 | 不开启 |
| enableItalics | **内置打印机**设置文本斜体 | 开启文本斜体功能 | 不开启 |
| enableInvert | 不支持设置文本倒置 | 不可用 | 无 |
| enAntiColor | 不支持设置文本反白 | 不可用 | 无 |


商米内置打印机默认使用矢量字体暂不支持自定义字体；

### 7. 打印绘制的内容

```java
void printCanvas(int count, PrintResult listener)

```

如果画布内容有效将指定打印机打印画布内容

**count 说明**

欲打印的次数，如果是标签打印机，将分别打印到每张标签上, 打印数量大于0张

**PrintResult 说明**

resultCode 打印成功返回0 打印失败返回 <0

message 失败时附加信息

商米内置打印机可以获取打印机的真实打印结果；对外接的打印机无法获得可考的打印结果；

## 四、综合示例

构建一个完整的由标题+二维码+文本+条码的样式模板，用于标签打印使用

```java
printer.canvasApi().initCanvas(
                    BaseStyle.getStyle()
                    .setWidth(330)
                    .setHeight(330));
printer.canvasApi().renderArea(
                    AreaStyle.getStyle()
                    .setStyle(Shape.BOX)
                    .setPosX(0)
                    .setPosY(0)
                    .setWidth(330)
                    .setHeight(330));
printer.canvasApi().renderText("这是标题",
                    TextStyle.getStyle()
                    .setPosX(165)
                    .setPosY(5)
                    .setAlign(Align.CENTER));
printer.canvasApi().renderQrCode("123456789",
                    QrStyle.getStyle()
                    .setPosX(5)
                    .setPosY(30).
                    setWidth(130)
                    .setHeight(130));
printer.canvasApi().renderText("商米是一家以“利他心”为核心价值观，" +
                            "全面引领全球智能商用硬件创新的物联网科技公司。",
                    TextStyle.getStyle()
                    .setPosX(140)
                    .setPosY(30)
                    .setWidth(180)
                    .setHeight(120));
printer.canvasApi().renderBarCode("12345678",
                    BarcodeStyle.getStyle()
                    .setPosX(15)
                    .setPosY(180)
                    .setWidth(300)
                    .setHeight(130));
printer.canvasApi().printCanvas(1, null);

```

## 五、打印标签技巧

### 1. 打印一张空白标签

当需要打印一张测试空白标签时可通过如下方式实现

```java
printer.canvasApi().initCanvas(
                    BaseStyle.getStyle()
                    .setWidth(330)
                    .setHeight(330));
printer.canvasApi().printCanvas(1, null);

```

### 2. 需要处理标签待取走的状态

在一些自带标签打印设备上如S2LCC机型支持检测标签纸是否未取走，因为未取走状态不会影响标签连续打印的进度，此时业务软件可以根据需要通过代码控制检测标签取走的情况

如果不关心标签是否取走的情况，就和一般打印方式一样通过printCanvas(n, null)一次可以打印N张标签

如果需要处理标签是否取走（只有取走后才可以打印下一张标签的业务场景）可通过如下代码实现

```java
... 绘制标签内容
printer.canvasApi().printCanvas(1, new PrintResult() {
                @Override
                public void onResult(int resultCode, String message) throws RemoteException {
                    监听打印标签的结果，当结果异常时通过queryApi获取打印机的实时状态，根据返回状态实现不同的业务需求
                    如标签待取走状态，可在此提示使用者及时拿走标签
                    当结果非异常可以决定是否打印下一张标签
                }
            );

```

打印机标签取走以及状态的恢复均可通过监听：woyou.aidlservice.jiuv5.NORMAL_ACTION 广播获取

### Symbology

条形码类型

| 枚举内容 | 说明 |
| --- | --- |
| UPCA | UPC-A条码商品条码是纯数字,位数是11位 在编码过后外加一位校验码，组成12位数字,主要在美国和加拿大使用 |
| UPCE | UPC-E条码商品条码是纯数字，是由UPC-A缩减而成，位数是7位，而且首位必须为0 在编码过后外加一位校验码，组成8位数字 |
| EAN13 | EAN13商品条码是纯数字，而且位数是12位 在编码过后外加一位校验码，组成13位数字 |
| EAN8 | EAN8商品条码是纯数字，而且位数是7位 在编码过后外加一位校验码，组成8位数字 |
| CODE39 | Code39条码生成字符集包括数字 、大写字母以及- . $ / + % * 空格等字符 其中"*"只用于标记开始和结束 |
| ITF | 交叉25码（Interleaved 2 of 5）条码生成，常用于物流管理 字符集仅为数字且个数为偶数,为奇数将自动在前面加"0" |
| CODABAR | 库德巴码（Codabar）条码生成，字符集包括数字和- $ : /. + 以及ABCD等字符 其中ABCD只用于开始或者结尾，作为标识符使用 |
| CODE93 | Code93条码生成是 full ASCII 模式，可使用ASCII全部128个字符 |
| CODE128 | 组合code128a、code128b、code128c，需根据码内容动态切换 |


### ErrorLevel 二维码纠错等级

| 枚举内容 | 说明 |
| --- | --- |
| L | 纠错等级L 7% |
| M | 纠错等级M 15% |
| Q | 纠错等级Q 25% |
| H | 纠错等级H 30% |


### ImageAlgorithm 图片转换算法

| 枚举内容 | 说明 | 参考浮动值 |
| --- | --- | --- |
| BINARIZATION | 二值化算法 | 二值化算法通过调整浮动值将转换不同彩色值为黑色 可根据图片颜色信息选择一般浮动值200 |
| DITHERING | 抖动灰度算法 | 抖动灰度算法不用考虑浮动值变化 |


### Shape 特殊图形

| 枚举内容 | 说明 |
| --- | --- |
| RECT_FILL | 填充矩形 矩形区域为黑块 |
| RECT_WHITE | 擦除矩形 矩形区域为白块 |
| RECT_REVERSE | 反白矩形 矩形区域将反白 |
| BOX | 空心矩形 |
| CIRCLE | 空心圆形 |
| OVAL | 空心椭圆 |
| PATH | 线段 |
